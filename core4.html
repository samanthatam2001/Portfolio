Variables A variable is a named item that changes or stores values. Ex: In a pie, the filling of the pie is the variable that can be changed to pumpkin, strawberry, apple, etc. Python: i < 9
Loops A loop is a sequence of instructions that is continually repeated until a certain condition is reached. Ex: Mickey Mouse made the broom keep adding water to the tub. Python: For square in range(4): square = forward(100) turn(90) print (square)
Conditionals Conditional statements, conditional expression, and conditional constructs are features of a programming language, which perform different computations or actions depending on whether a programmer-specified boolean condition evaluates to true or false. Ex: If broom alive: then walk away, else: try again. Python: If i in alphabet: print (alphabet) else: print ("Not here")
Functions A chunk of code that is reused to make the same thing happen.
